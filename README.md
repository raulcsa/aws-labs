# AWS Labs ☁️

Bienvenido a mi repositorio de prácticas con Amazon Web Services (AWS).

Aquí voy a ir documentando los laboratorios que realizo mientras aprendo a usar los principales servicios de AWS: redes, bases de datos, almacenamiento, seguridad y despliegue de aplicaciones web.

## ¿Qué voy a hacer?

- Crear y configurar redes virtuales (VPC) en AWS
- Desplegar bases de datos administradas con Amazon RDS
- Usar almacenamiento compartido con Amazon EFS
- Gestionar identidades y permisos con AWS IAM
- Desplegar aplicaciones web dinámicas sobre EC2

Cada carpeta o archivo de este repositorio corresponde a un laboratorio práctico con capturas de pantalla y los pasos seguidos.
